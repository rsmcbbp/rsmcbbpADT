使用前阅读：

    本项目的成果主要分为三个部分：①各类卫星云图指标函数；②TYIntensity函数；③可视化界面GUIDE。

    所使用的云图材料必须为agora的“赤外1”云图。运行时，注意将图片文件名保存为可供MATLAB读取的变量名（以英文为主，不含中文字符，不含大多数标点符号），并将文件添加进当前文件夹或路径中，方能正常运行。

    大多数卫星云图指标函数（包括台风强度估算的最终公式，即TYIntensity函数）的格式相同，为function xxx=fname(name,year)。这些函数都可以直接调用，返回的值就是该主观指标的测算值。在输入name项时，要带上单引号，即’name’。year项最好输入，如果不输入，系统默认为2100，即以2005年以后的公式进行计算。在运行MainScript.m与GUIDE.m时，只要files文件夹在MATLAB路径中，那么所有部分就可以正常运转。对于每个台风，name与year两项的输入要准确。

    若要直接体验本项目的成果，可以直接运行可视化界面GUIDE。该界面的操作流程如下：①选择台风云图文件，随后界面左侧窗口将显示台风的原始IR-BW云图；②勾选台风活动的年份选项，若错误勾选将导致分析失真；③点击计算按钮进行运算，右侧窗口将显示转换后的IR-BD云图，窗口下方将显示计算结果；④要重新计算其它台风的强度，只需要重复以上步骤即可。
